KanTan

What is this?

This tool is designed to make kanji lookup easier by combining kanji lookup and word lookup into one. Instead of finding each kanji separately and then looking up the complete word in the dictionary, here you can specify a pattern for a complete word using the radicals that you recognize, and find all the words that match that pattern.

Syntax and usage

  • A pair of brackets matches a single kanji. Between the brackets you may specify the radicals that the kanji must contain.
    • [罒]す matches 壊す
    • [儿田]法 matches 魔法
    • 真っ[]ぐ matches 真っ直ぐ
    • [韭][火] matches 殲滅
    You can insert radicals using the radical table. Brackets are added automatically if necessary.
  • You can use lowercase romaji for hirgana and uppercase romaji for katakana.
    • []BO-RU matches 段ボール
    • [冖]biru matches 帯びる
  • a dot (. or 。) stands for a single hiragana character. This is useful when you don't know the dictionary form of a word.
    • [已]. matches 選ぶ
